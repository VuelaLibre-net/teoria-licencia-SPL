# Código de conducta

Este proyecto existe para mejorar materiales de formación aeronáutica para pilotos de planeadores en España. Queremos una comunidad rigurosa, amable y centrada en la seguridad operacional.

## Comportamiento esperado

- Tratar a las demás personas con respeto.
- Criticar ideas, textos y cambios concretos, no a las personas.
- Aportar fuentes cuando se discutan cuestiones técnicas o normativas.
- Aceptar que una corrección puede requerir revisión editorial antes de integrarse.
- Mantener las conversaciones enfocadas en mejorar los manuales.

## Comportamiento no aceptado

- Insultos, ataques personales o lenguaje discriminatorio.
- Acoso, intimidación o contacto no deseado.
- Publicar información privada de otras personas sin permiso.
- Introducir deliberadamente información falsa o peligrosa.
- Convertir una discusión técnica en una disputa personal.

## Aplicación

Los mantenedores pueden editar, ocultar o cerrar comentarios, issues y Pull Requests que incumplan este código de conducta. También pueden bloquear temporal o permanentemente a quienes mantengan comportamientos incompatibles con una colaboración sana.

## Contacto

Si quieres informar de un problema de conducta, escribe a `info@vuelalibre.net`. Incluye contexto suficiente para poder revisar lo ocurrido.
